# 🎨 텍스트 이펙트 생성기

폰트와 효과를 선택하여 텍스트를 PNG 이미지로 내보낼 수 있는 웹 애플리케이션입니다.

## 주요 기능

- **텍스트 입력**: 원하는 텍스트를 입력하세요
- **폰트 선택**: 10가지 인기 Google Fonts 지원
- **폰트 크기 조절**: 24px ~ 200px 범위에서 자유롭게 조절
- **색상 설정**: 텍스트 및 배경 색상 변경
- **투명 배경**: 투명한 배경 이미지 생성 가능
- **그림자 효과**: 색상, 블러, 오프셋 조절
- **테두리 효과**: 색상 및 두께 조절
- **실시간 미리보기**: 변경 사항 즉시 확인
- **PNG 다운로드**: 설정한 텍스트 이펙트를 PNG로 다운로드

## 기술 스택

- **React 18**: UI 프레임워크
- **TypeScript**: 타입 안정성
- **Vite**: 빠른 개발 환경
- **Canvas API**: 텍스트 렌더링

## 설치 및 실행

### 요구사항

- Node.js 14 이상 (권장: 18 이상)
- npm 또는 yarn

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

### 빌드

```bash
npm run build
```

### 미리보기

```bash
npm run preview
```

## 사용 방법

1. **텍스트 입력**: 상단 입력 필드에 원하는 텍스트를 입력하세요
2. **폰트 선택**: 드롭다운에서 원하는 폰트를 선택하세요
3. **효과 설정**: 왼쪽 패널에서 폰트 크기, 색상, 그림자, 테두리 등을 조절하세요
4. **미리보기 확인**: 오른쪽에서 실시간 미리보기를 확인하세요
5. **다운로드**: "PNG 다운로드" 버튼을 클릭하여 이미지를 저장하세요

## 프로젝트 구조

```
src/
├── components/          # UI 컴포넌트
│   ├── TextInput.tsx    # 텍스트 입력
│   ├── FontSelector.tsx # 폰트 선택
│   ├── EffectPanel.tsx  # 효과 설정 패널
│   ├── Canvas.tsx       # 미리보기 캔버스
│   └── ExportButton.tsx # PNG 다운로드
├── types/               # TypeScript 타입 정의
├── utils/               # 유틸리티 함수
│   ├── canvas.ts        # Canvas 렌더링
│   └── export.ts        # PNG 내보내기
├── App.tsx              # 메인 앱
└── main.tsx             # 진입점
```
